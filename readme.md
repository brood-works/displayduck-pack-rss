## About
This pack contains an RSS feed widget for DisplayDuck.

<p align="right">(<a href="#readme-top">back to top</a>)</p>


## Getting Started
Configure a feed URL and the widget will fetch and render the latest feed entries.


<p align="right">(<a href="#readme-top">back to top</a>)</p>

---

## Configurable options
| Setting | Type | Configurable Values | Default Value
|---|---|---|---|
| Feed URL | `string` | any valid RSS URL | ``
| Refresh Interval (minutes) | `number` | any `number` >= 0 | `5`
| Max Items | `number` | `1` to `5` | `5`
| Skip Items | `number` | any `number` >= 0 | `0`
